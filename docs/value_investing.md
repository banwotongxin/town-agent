# 价值投资原则

## 核心理念

价值投资的核心理念是寻找被市场低估的股票，以低于其内在价值的价格买入，并长期持有。

### 1. 能力圈原则

只投资自己理解的行业和公司。巴菲特强调，投资者应该在自己的"能力圈"内行动，不要盲目跟风。

**关键要点：**
- 了解公司的商业模式
- 理解行业的竞争格局
- 能够预测公司未来5-10年的发展

### 2. 安全边际

以显著低于内在价值的价格买入，为判断错误留出缓冲空间。

**计算方法：**
- 市盈率（P/E）低于行业平均
- 市净率（P/B）小于1.5
- 股息率高于债券收益率

### 3. 长期持有

价值投资不是短期投机，而是长期持有优质公司。

**优势：**
- 减少交易成本
- 享受复利效应
- 避免情绪化决策

## 经典案例

### 可口可乐

巴菲特在1988年买入可口可乐，持有至今超过30年，获得了超过1000%的回报。

**成功原因：**
- 强大的品牌护城河
- 稳定的现金流
- 全球市场需求

### 苹果公司

2016年开始大量买入苹果，成为伯克希尔最大持仓。

**投资逻辑：**
- 生态系统锁定用户
- 高利润率
- 强大的创新能力

## 风险提示

1. **价值陷阱**：看似便宜但基本面恶化的公司
2. **时间成本**：可能需要多年才能体现价值
3. **市场非理性**：价格可能长期偏离价值
